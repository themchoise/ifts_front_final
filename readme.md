# Trabajo Práctico Final Integrador

## Información del Curso
- **Materia:** Desarrollo de Sistemas web FrontEnd – IFTS 18
- **Cursada:** Primer Cuatrimestre 2025

## Objetivo
Se debe desarrollar una **Landing Page responsive** para Mobile y Desktop haciendo uso de:
- Tags HTML5
- Clases CSS
- CSS Grid
- FlexBox
- Custom Properties
- Media queries

## Estructura de Archivos
La carpeta solución del TP debe tener el nombre y apellido del alumno y debe contener los siguientes archivos:

- **`index.html`** - Ubicado en la raíz de la solución
- **`images/`** - Subcarpeta que contenga los archivos de imágenes del classroom
- **`css/`** - Subcarpeta que contenga el archivo `style.css` con los estilos

## Aspectos Generales a Tener en Cuenta

### Validación de Código
- **HTML:** DEBE VALIDAR según el [validador W3C](https://validator.w3.org/#validate_by_input)
- **CSS:** DEBE VALIDAR según el [validador W3C](https://jigsaw.w3.org/css-validator/#validate_by_input)
- **NO** está permitido agregar contenido CSS con el atributo `style` dentro del HTML

### Diseño y Recursos
- Las imágenes del diseño se encuentran en la carpeta "design" del classroom
- **Fuente:** [League Spartan](https://fonts.google.com/specimen/League+Spartan)
  - **Weights:** 500, 600, 700
- **Tamaño de fuente base:** 12px para el `body`

### Breakpoints de Diseño
1. **Mobile:** 375px
2. **Desktop:** 1440px

### Paleta de Colores (Custom Properties)
1. **Dark Gray:** `hsl(0, 0%, 63%)`
2. **Black:** `hsl(0, 0%, 0%)`
3. **White:** `hsl(0, 0%, 100%)`
4. **Very Dark Gray:** `hsl(0, 0%, 27%)`
